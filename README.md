# How to generate documentation

This documentation has been developed to work in MkDocs. To render it locally, do the following:

- install MkDocs python package
- cd into the `dig-docs` folder
- from shell (e.g. WSL) launch the server: `$ mkdocs serve`
- Doc pages will be locally rendered opening the URL: http://127.0.0.1:8000/ in a browser

Documentation on how to use `MkDocs` can be found [here](https://www.mkdocs.org/)

   - DAQ Parameters: setup/parameters.md
   - Pedestal: setup/pedestal.md
   - Calibrator: setup/calibrator.md
